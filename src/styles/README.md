# Estilos (CSS)

Aquí se concentran los archivos de estilo del frontend.

- **globals.css** — Estilos globales, variables CSS, utilidades Tailwind.

El resto del diseño se hace con **Tailwind** en los componentes (`src/app/`, `src/components/`).  
Breakpoints y tema: **tailwind.config.ts** en la raíz del proyecto.
